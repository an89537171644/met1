# Архитектура

## Основной поток данных

```text
геометрия + нагрузки + сортамент
        ↓
нагрузочные случаи и сочетания
        ↓
расчетная модель / импорт ЛИРА
        ↓
усилия, перемещения, проверки
        ↓
подбор единого сечения
        ↓
ML top-N рекомендация
        ↓
обязательная проверка
        ↓
отчет
```

## Планируемые модули

- `geometry.py`: геометрия рамы, узлы, элементы, уклон кровли.
- `materials.py`: марки стали и нормативные свойства.
- `sections.py`: сортамент и характеристики профилей.
- `sp20_loads.py`: нагрузочные случаи по СП 20.
- `load_combinations.py`: сочетания нагрузок.
- `lira_import.py`: импорт экспортов ЛИРА.
- `ml_data.py`: подготовка датасета.
- `ml_model.py`: обучение и прогнозирование.
- `checks_sp16.py`: инженерные проверки по СП 16.
- `optimizer.py`: перебор и выбор минимального проходящего сечения.
- `reports.py`: отчеты Markdown/CSV.
- `ui_streamlit.py`: пользовательский интерфейс MVP.

## Ограничения MVP

- Узлы, сварные швы, болты, база колонны, фундаменты и пространственная работа здания не подбираются.
- ML не заменяет нормативный расчет.
- Инженерные проверки по СП 16 должны считаться неполными, пока не пройдут отдельную верификацию.
