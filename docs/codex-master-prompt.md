Ты работаешь над магистерским проектом: программа подбора единого сечения для плоских двухскатных однопролётных стальных рам.

Принятые условия:
- расчёт стальных конструкций: СП 16.13330.2017;
- нагрузки и сочетания: СП 20.13330.2016;
- объект: плоская двухскатная однопролётная рама;
- единое сечение применяется к стойкам и наклонным ригелям;
- обучающие данные должны импортироваться из расчётов ЛИРА-САПР / ПК ЛИРА;
- нейросеть должна предлагать top-N сечений, но результат должен проходить проверку;
- нужен CLI и пользовательский интерфейс, для MVP использовать Streamlit.

Перед работой изучи:

2. docs/sp20-loads.md
3. docs/ml-dataset.md
