# Бэклог работ

## Sprint 0: каркас проекта

- Создать структуру Python-проекта.
- Настроить `pyproject.toml`, `pytest`, `ruff`.
- Создать CLI-команды: `init`, `validate-input`, `build-loads`, `import-lira`, `build-dataset`, `train`, `predict`, `verify`, `report`, `ui`.
- Создать Pydantic-модели данных.
- Добавить тесты на загрузку конфигурации и валидацию геометрии.
- Перенести документы и CSV-шаблоны.

## Sprint 1: геометрия и сортамент

- Реализовать модель плоской двухскатной рамы.
- Вычислять уклон кровли по пролету и высотам.
- Проверять базовую корректность геометрии.
- Реализовать загрузку сортамента из CSV.
- Проверять правило единого сечения для стоек и ригелей.

## Sprint 2: нагрузки по СП 20

- Реализовать структуру нагрузочных случаев.
- Поддержать постоянные, снеговые, ветровые и ремонтные нагрузки.
- Реализовать генерацию базовых load cases.
- Реализовать таблицу сочетаний.
- Добавить валидацию единиц, коэффициентов и источников нормы.

## Sprint 3: импорт ЛИРА

- Импортировать CSV-таблицы из ЛИРА.
- Нормализовать единицы.
- Собрать `case_id`, усилия, перемещения, проверки, выбранные сечения.
- Создать отчет качества импорта.
- Исключать неполные или противоречивые кейсы.

## Sprint 4: ML-датасет

- Сформировать признаки.
- Сформировать целевые метки.

## Sprint 5: baseline-модель

- Обучить простую baseline-модель.
